Reproductor Web Espotify

Sistema de gestión centralizada que consta de dos proyectos interrelacionados: ServidorCentralWeb y ServidorWeb. ServidorCentralWeb gestiona la lógica de negocio, la persistencia de datos y expone un servicio web SOAP, mientras que ServidorWeb es una aplicación web que consume este servicio, proporcionando una interfaz para los usuarios finales. Juntos, permiten la gestión y visualización de datos de manera eficiente.

Java Html Css JavaScript MySql

Es una aplicación diseñada para administrar las reservas en un hostel. El sistema permite gestionar los huéspedes, las habitaciones, y las reservas, con una estructura de precios y descuentos según el tipo de reserva.

Gestión de Mangas

Es una aplicación de consola en Python que permite gestionar mangas utilizando operaciones CRUD (Crear, Leer, Actualizar, Eliminar) con una base de datos MySQL. El sistema permite agregar nuevos mangas, actualizar su información, eliminar mangas existentes y listar los mangas registrados en la base de datos.
